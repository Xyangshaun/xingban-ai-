01 / AI 对话

懂情绪
有性格
会倾听

虚拟伙伴能识别 8 种情绪，根据 5 种性格给出不同风格的回应，还会记住你说过的话，在后续对话中自然引用。不是模板回复，是真正的对话。

  • 情绪识别8 种
  • 性格风格5 种
  • 上下文记忆支持
  • AI 引擎通义千问 API

02 / AR 空间

伙伴就在
你面前

利用 PICO 空间计算能力，将虚拟伙伴放置在真实空间中。你可以走动、转头、从不同角度看它。这种存在感，是手机屏幕无法提供的。

  • 空间锚定PICO Spatial SDK
  • 追踪方式六自由度
  • 交互方式空间控制器
  • 场景模式XR 全沉浸

05 / 语音克隆

用熟悉的
声音说话

  • 语音框架Sherpa-onnx
  • TTS 模型VITS 中英双语
  • 克隆时长30 秒
  • 运行方式完全离线

技术架构

前沿技术
三位一体

PICO 空间计算 + AI 大模型 + 离线语音合成，构建从感知到交互到反馈的完整技术闭环。

Platform
PICO Spatial SDK
VR 空间计算框架，提供六自由度追踪、空间锚定和环境理解
AI Model
通义千问 API

Design

Language
Kotlin
